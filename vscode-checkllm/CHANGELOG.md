# Changelog

## 0.1.0 (2026-04-06)

Initial release.

- CodeLens annotations above test functions with pass/fail, avg score, cost
- Gutter decorations with green/red dots
- Hover popup with per-check score table
- File watcher for auto-refresh on `.checkllm/` changes
- Commands: Run, Estimate, Refresh
- Configurable judge backend, model, threshold, and budget
- Status bar showing total check results
