# Changelog

All notable changes to the checkllm VS Code extension will be recorded here.
This project follows [Semantic Versioning](https://semver.org/).

## 0.1.0 — Initial release

- Commands: `checkllm.runTest`, `checkllm.runSuite`, `checkllm.dashboard`,
  `checkllm.refreshTree`.
- Tree view listing `*.checkllm.yaml` files in the workspace.
- Status bar item showing the latest pass/fail count.
- Language contribution for `*.checkllm.yaml` with metric-aware
  syntax highlighting.
